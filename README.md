# Employee Leave Management System (ELMS)

## Giới thiệu
Employee Leave Management System (ELMS) là hệ thống web hỗ trợ nhân viên gửi đơn nghỉ phép và hỗ trợ quản lý xét duyệt đơn nghỉ phép một cách thuận tiện và minh bạch.

## Mục tiêu MVP
- Đăng nhập / đăng xuất
- Tạo đơn nghỉ phép
- Xem danh sách đơn nghỉ phép
- Xem chi tiết đơn nghỉ phép
- Phê duyệt / từ chối đơn nghỉ phép
- Dashboard tổng quan
- Xem số ngày phép còn lại

## Công nghệ sử dụng
- Frontend: HTML, CSS, Bootstrap
- Backend: Java Servlet
- ORM/Persistence: JPA
- IDE: Spring Tool Suite 4
- Database: SQL Server
- Version Control: Git + GitHub
- Project Management: Jira
- UI Design: Figma / Balsamiq

## Cấu trúc thư mục
